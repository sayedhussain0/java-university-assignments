Write a program that reads records for multiple students. Each record consists of first name (string)
followed by his/her GPA (double). Assume the number of records is unknown and the program stops
reading when the word ‘stop’ is entered as the student’s name. The program then displays on screen
the following:

(a) The number of students named AHMED.

(b) The number of students whom GPAs are above 3.0.

(c) The name that has the maximum GPA.

********************************************************************************
Sample Input/Output:

Enter students GPA:

ALI 3.5

FATIMA 2.1 

MARYAM 4.0 

AHMED 2.9

stop

Number of students with name AHMED = 1 

Number of students with GPA above 3.0 = 2 

The maximum GPA is MARYAM
